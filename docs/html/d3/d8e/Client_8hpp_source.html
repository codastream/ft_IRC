27 *
28 * This macro is used as a bitmask flag to indicate that a client
29 * has not yet been authenticated on the server.

33/**
34 * @def AUTHENTICATED
35 * @brief Represents the authenticated state of a client.

37 * This macro is used as a bitmask flag to indicate that a client
38 * has been authenticated on the server after sending a successful PASS.
39 * @see #Pass
40 * - Value: `0b00000` (binary 00001, decimal 1)
